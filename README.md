# Python 爬虫 & Flask 项目集合
自己学习用的Python爬虫、Flask Web开发练习项目

## 项目内容
- Flask 基础Web服务（hello.py / Flask_project.py）
- 日本招聘网站爬虫练习
- 环境配置：Anaconda + PyCharm

## 技术栈
- Python 3.x
- Flask
- requests / BeautifulSoup（爬虫）
- Anaconda 虚拟环境

## 运行说明
1.  确保已安装Anaconda，激活 `spider_env` 环境
2.  运行 `Flask_project.py` 启动Web服务
3.  访问 `http://127.0.0.1:5000` 查看效果
